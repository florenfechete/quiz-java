# Quiz en Java

Proyecto de prácticas para la asignatura de Programación en Java.

## Funcionalidades
- [ ] Menú interactivo por consola
- [ ] Preguntas de tipo verdadero/falso y múltiple opción
- [ ] Sistema de puntuación
- [ ] Manejo de excepciones
- [ ] Interfaz gráfica (Swing)

## Estructura del proyecto
```
quiz-java/
├── src/
│   ├── Main.java
│   ├── Pregunta.java
│   └── Quiz.java
└── README.md
```

## Temas practicados
- Tema 2: Clases, objetos y métodos
- Tema 3: Herencia
- Tema 4: Programación dirigida por eventos
- Tema 5: Excepciones
- Tema 6: Colecciones
```
